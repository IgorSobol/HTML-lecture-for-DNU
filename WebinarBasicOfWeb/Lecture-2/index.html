каскадные таблицы стилей

         *****
       }

Синтаксис

Синтаксис

       заголовок   {

       }

Примеры селекторов:

       div {...}
       .logo {...}
       div.logo {...}
       .logo img {...}

Cелекторы читают справа налево:

       div.card .card-item a img {...}

Вложенность определяется пробелом

       div.card   .card-item   a   img {...}

Псевдоклассы:

       :link {...}
       :hover {...}
       :active {...}
       :focus {...}
